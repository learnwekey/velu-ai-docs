Use this guide to add a reusable skill in Application Settings so your team can apply consistent prompts across workflows.

## Before You Begin

Complete these checks before you start:

- Make sure you're able to open **Settings** in your application.
- Confirm that the **Skills** tab is available to you.

## Steps

Follow these steps to create the skill:

### 1. Navigate to Skills Settings

1. Open your application and go to **Settings**.
2. In the left navigation, select **Skills** to open the skills list.

![Skills settings page showing the Skills tab selected](/img/skills-page-in-application-settings-00-00-01-95dce620.png)

### 2. Start a New Skill

Click **+ Add Skill** in the top-right corner of the Skills page to open the creation dialog.

![Cursor clicking the Add Skill button on the Skills page](/img/clicking-the-add-skill-button-00-01-02-71b2bb36.png)

### 3. Define the Skill Details

In the **Create New Skill** window, fill in each field:

- **Name** — Provide a concise name such as **Blog Writing Skill**.
- **Description** — Summarize how teammates should use this skill.
- **Content** — Add the full prompt or instructions the artificial intelligence (AI) should follow.

![Create new skill modal with Name, Description, and Content fields completed](/img/filling-out-the-new-skill-form-with-name-description-and-content-01-18-03-9391b9b7.png)

### 4. Save and Confirm

1. Click **Save** to store the new skill.
2. Confirm the skill appears in the list and is ready for use in other workflows.

![Save button highlighted in the Create new skill dialog](/img/saving-the-new-skill-01-21-04-7abd4467.png)

![Skills list showing the newly created skill](/img/the-newly-created-skill-is-now-visible-in-the-skills-list-01-22-05-b259922c.png)

## Outcome

Your custom skill is now available in the Skills list, and it appears wherever the application lets you choose a predefined skill.
