# Setup

This page will help you set up Velu AI for your documentation repo.

## 1. Prerequisites

Before you start, youâ€™ll need:

- A GitHub account
- A documentation repository (for example, your Docusaurus docs repo)
- Access to Velu AI (your dashboard URL and login)


## 2. Sign up with Google

1. On the Velu AI login page, select **Sign up with Google**.
2. Choose the Google account associated with your Velu AI workspace, or add a new account.
3. Review the requested permissions (profile and email) and click **Allow**.
4. Confirm the workspace you want to join or create a new workspace if prompted.
5. After the redirect completes, your Google identity is linked to Velu AI and you can proceed to connect integrations.

## 3. Connect your GitHub account

1. Log in to the Velu AI dashboard.
2. Go to **Integrations** in the left menu.
3. Click **GitHub**.
4. Click **Connect GitHub** and authorize Velu AI.
5. Select the GitHub organization or user whose repos you want Velu AI to access.

## 4. Add your documentation repository

1. In Velu AI, go to **GitHub-Docs**.
2. Select the repo that contains your docs (for example, your Docusaurus repo).
3. Choose connect repo.



Next, read the [Integrations](./integrations.md) page to understand what tools Velu AI can connect with.
